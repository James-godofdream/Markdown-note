# 职场成功的两条腿
## 硬实力：技术能力
#### 1、职场的基础
#### 2、小白重视
#### 3、难以短时间提高
#### 4、硬实力更多的是影响工作
## 软实力：人际沟通、协作、管理能力
#### 1、职场晋升的关键
#### 2、观念认为天生不可改变，大多数忽视训练
#### 3、短时间可以迅速提高
#### 4、软实力，影响工作和人生的方方面面

#### 1、基础实力（形象气质、声音魅力）---人生

#### 3、领导能力（影响力、领导力、情商）--高层
